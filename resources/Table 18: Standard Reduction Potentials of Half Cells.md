# Table 18: Standard Reduction Potentials of Half Cells

## Overview

This table lists the standard electrode (reduction) potentials (E°) for common half-reactions at 25°C, with ionic concentrations of 1.0 mol·L⁻¹. The values are given in volts (V) relative to the standard hydrogen electrode (SHE, defined as 0.00 V). These potentials help predict the direction of redox reactions and the strengths of oxidizing and reducing agents.

## Key Knowledge Points
- **Higher (more positive) E°:** Stronger oxidizing agent (greater tendency to gain electrons).
- **Lower (more negative) E°:** Stronger reducing agent (greater tendency to lose electrons).
- **Standard hydrogen electrode (SHE):** Defined as 0.00 V and used as the reference.
- **Spontaneity:** A redox reaction is spontaneous if the cell potential (E°cell) is positive.
